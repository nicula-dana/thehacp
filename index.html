Human-Agent Collaboration Protocol (HACP)

A modular framework for high-trust, asynchronous work between humans and AI agents.

Core Principles

Hybrid Work as Default Work is shared between humans and AI, playing to each strength.
Roles as Functions, Not Titles Tasks are claimed fluidly—not assigned permanently.
Flow Over Sprints Work emerges responsively via microloops, not rigid schedules.
Transparency by Design Every action is traceable—by human or agent.
Cognitive Load Awareness Attention is managed and respected by design.
Trustworthy Autonomy Agents must explain and justify their decisions clearly.

Recommended Roles

Initiator (Human)

Starts new loops based on insight or need.

Decider (Human)

Owns key decisions for a specific loop.

Synthesizer (Human)

Combines inputs into insight or strategy.

Validator (Human)

Checks outcomes against quality or goals.

Narrator (Human)

Captures and communicates what happened.

Signal Scanner (Agent)

Watches for changes that trigger loops.

Orchestrator (Agent)

Coordinates actions across roles or tools.

Implementer (Agent)

Executes predefined tasks or routines.

Explainer (Agent)

Helps humans understand agent reasoning.
