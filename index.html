Overview

HACP (Human-Agent Collaboration Protocol) is a modular, open protocol designed to enable effective, high-trust collaboration between humans and AI agents in remote, asynchronous, and dynamic work environments. It redefines traditional agile frameworks to accommodate fluid roles, real-time insights, and machine-human co-creation.

Core Principles

  • Hybrid Work as Default
  • Roles as Functions, Not Titles
  • Flow over Sprints
  • Transparency by Design
  • Cognitive Load Awareness
  • Trustworthy Autonomy

Protocol Structure

1. Signals

Discrete triggers that initiate collaboration loops.

2. Loops

Short, goal-oriented collaboration cycles.

3. Role Tokens

Claimable responsibilities such as: Decider, Synthesizer, Validator, etc.

4. Artifacts

Outputs include SAGs, Priority Graphs, and Shadow Logs.

5. Boundaries

Constraints: Ethical, Cognitive, Technical.

Advanced Concepts

  • Dynamic Role System
  • Agent-Oriented Workflows (AOWs)
  • Cognitive Load Contracts (CLCs)
  • Trust Ledger

Implementation Guide

  • Start Small
  • Visualize Everything
  • Log Transparently
  • Evaluate Loops
  • Scale Modularly
