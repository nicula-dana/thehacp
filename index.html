Overview

HACP (Human-Agent Collaboration Protocol) is an open specification for hybrid teams composed of humans and AI agents working asynchronously, fluidly, and with clear accountability.

Core Principles

Hybrid Work as Default

Humans and agents collaborate as peers with complementary skills.

Roles as Functions, Not Titles

Responsibility is claimed per loop, not assigned by hierarchy.

Flow over Sprints

Work adapts to real-world conditions via microloops.

Transparency by Design

Actions and decisions are visible, logged, and auditable.

Cognitive Load Awareness

Agent behavior adjusts based on user attention budgets.

Trustworthy Autonomy

Agents earn influence by being explainable and consistent.

Role Tokens

Initiator (Human)Proposes new work
Decider (Human)Makes final decisions
Synthesizer (Human)Integrates ideas
Validator (Human)Assesses quality
Narrator (Human)Captures and explains
Signal Scanner (Agent)Watches for triggers
Orchestrator (Agent)Coordinates collaboration
Implementer (Agent)Executes tasks
Logger (Agent)Logs actions
Explainer (Agent)Makes logic visible

Collaboration Objects

  • SAG: Situational Alignment Graph
  • Priority Graph: Live priority flow
  • Shadow Logs: Agent memory trails

Advanced Concepts

Microloops

Every output is a cycle of sense–align–act–reflect.

Dynamic Role System

Tokens can be claimed or passed based on trust and context.

Trust Ledger

Track confidence in agent behaviors over time.

CLCs – Cognitive Load Contracts

Human participants define how much attention they can give.
